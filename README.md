# Real-Time-Driver-Drowsiness-and-Emotional-Expression-Recognition-Using-YOLO-on-Edge-Devices
This study presents an experimental analysis of real-time driver drowsiness and emotional expression recognition using a YOLOv8n-classification model trained on a live-collected dataset and deployed on two computing platforms: a Host Computer equipped with an NVIDIA RTX Studio CPU and a resource-constrained NVIDIA Jetson Nano CPU edge device
